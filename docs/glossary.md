# Glossary

このドキュメントには、プロジェクト内で使う用語の意味や表記を整理する。

用語の解釈が曖昧な場合や、ドキュメント間で表記が揺れている場合は、このドキュメントを更新する。

## 用語

| 用語 | 意味 | 備考 |
| --- | --- | --- |
| Expo | React Native アプリケーションの開発、ビルド、配布を支援するフレームワークとサービス群 | 採用方針は `docs/architecture/tech-stack.md` を参照する |
| 散歩セッション | 散歩中の作業や記録をまとめる単位 | 詳細は仕様策定時に定義する |
| 母艦の Mac | Codex CLI と開発環境を持つ接続先の Mac | iPhone アプリから SSH で接続する |
| tmux | ターミナルセッションを保持、分割、再接続できるツール | Codex CLI の実行セッション維持に使う |
| Tailscale | WireGuard ベースの VPN サービス | このプロジェクトでは前提インフラとして扱い、実装対象外とする |
| TTS | Text-to-Speech の略で、テキストを音声に変換する仕組み | irodori-TTS などを候補にする |
